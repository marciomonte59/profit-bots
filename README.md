# ProfitBots — Trading Bots

Bots de análise de mercado para Telegram hospedados no Railway.

## Bots
- @ProfitAnalise_bot — Análise de gráficos com IA
- @ProfitSinal_bot — Sinais de trading
- @ProfitEstrategia_bot — Repositório de estratégias
- @ProfitSentiBot — Sentimento do mercado

## Deploy
Railway + Python 3.11
